# Nusalingo Kids Project Documentation

## Overview
Nusalingo Kids is an engaging educational platform designed to help children learn languages through fun and interactive methods.

## Features
- Interactive lessons tailored for kids
- Games and quizzes for language reinforcement
- Progress tracking for both kids and parents
- Support for multiple languages

## Getting Started
To get started with Nusalingo Kids, follow these steps:
1. Clone the repository: `git clone https://github.com/ushan67/nusalingo-kids-pro.git`
2. Navigate to the project directory: `cd nusalingo-kids-pro`
3. Install the required dependencies: `npm install`
4. Start the application: `npm start`
